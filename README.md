# Мэдлэгийн Тулаан (Монгол Quiz) — Next.js + Vercel

## Ажиллуулах (локал)

```bash
npm install
npm run gen:questions
npm run dev
```

Дараа нь `http://localhost:3000` руу орно.

## Vercel дээр deploy хийх

- GitHub руу push хийнэ
- Vercel дээр Import → Deploy

## Одоогоор байгаа боломжууд

- 1000+ Монгол хэл дээрх асуултын сан (`public/questions.mn.json`)
- Өдрийн сорил (`/play`): тохиргоогүй, шууд тоглоно
- Тест дууссаны дараа үр дүн + хуваалцах
- Цуврал (streak) + Best (localStorage)

## Дараагийн алхам (дараа нь хийнэ)
